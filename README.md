# 🛡️ Spam Shield AI

A Machine Learning-based Email Spam Detection System with an integrated AI Cyber Assistant.

## 📌 Project Overview

Spam Shield AI detects spam emails using Machine Learning and provides cybersecurity guidance through an AI chatbot.

## ✨ Features

- 📧 Email Spam Detection
- 🤖 AI Cyber Assistant
- ⚡ Fast Predictions
- 🌐 User-Friendly Interface
- 🔒 Cybersecurity Support

## 🛠️ Technologies Used

- Python
- Flask
- Scikit-learn
- Pandas
- HTML
- CSS
- JavaScript
- Groq API

## 🚀 Installation

### Clone the repository

```bash
git clone https://github.com/radhikatalekar/Spam_Shield_AI.git
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run the project

```bash
python app.py
```

Open your browser and visit:

```
http://127.0.0.1:5000
```
